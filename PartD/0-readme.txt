Part D. Pretty simple program. When you enter the argument, the program will output the process you have chose.

example output:

>java NewProcess 0

Please choose a process from 1 - 3
Process: 0 is running

>java NewProcess 1

Please choose a process from 1 - 3
Process: 1 is running

>java NewProcess 2

Please choose a process from 1 - 3
Process: 2 is running
